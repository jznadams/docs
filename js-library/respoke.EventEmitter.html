Overview

A generic class for emitting and listening to events.

fire(eventType, evt)

Trigger an event on an object. All listeners for the specified eventType will be called. Listeners will be bound to the object ('this' will refer to the object), and additional arguments to fire() will be passed into each listener.

Arguments

Name
Type
eventType
string
A developer-specified string identifying the event to fire.

evt
string, number, object, array
Any number of optional parameters to be passed to the listener

hasListeners(eventType)

returns boolean

Determine if an object has had any listeners registered for a given event outside the library. This method checks for the isInternal flag on each listener and doesn't count it toward an event being listened to. This method is used in the library to handle situations where an action is needed if an event won't be acted on. For instance, if a call comes in for the logged-in user, but the developer isn't listening to {respoke.Client#call}, we'll need to reject the call immediately.

Arguments

Name
Type
eventType
string
The name of the event

ignore(eventType, listener)

Remove a listener from an object. If no eventType is specified, all eventTypes will be cleared. If an eventType is specified but no listener is specified, all listeners will be removed from the specified eventType. If a listener is also specified, only that listener will be removed.

Arguments

Name
Type
eventType
string
An optional developer-specified string identifying the event.

listener
function
An optional function to remove from the specified event.

listen(eventType, listener, isInternal)

Add a listener to an object. This method adds the given listener to the given event in the case that the same listener is not already registered to this even and the listener is a function. The third argument 'isInternal' is used only internally by the library to indicate that this listener is a library-used listener and should not count when we are trying to determine if an event has listeners placed by the developer.

Arguments

Name
Type
eventType
string
A developer-specified string identifying the event.

listener
respoke.EventEmitter.eventListener
A function to call when the event is fire.

isInternal
boolean
A flag to indicate this listener was added by the library. This parameter should not be used by developers who are using the library, only by developers who are working on the library itself.

once(eventType, listener, isInternal)

Add a listener that will only be called once to an object. This method adds the given listener to the given event in the case that the same listener is not already registered to this event and the listener is a function. The third argument 'isInternal' is used only internally by the library to indicate that this listener is a library-used listener and should not count when we are trying to determine if an event has listeners placed by the developer.

A function to call when the event is fire.
